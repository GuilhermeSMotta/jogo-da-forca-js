# Jogo da Forca (JS)
Jogo da forca feito em HTML, CSS e JavaScript.
## Como iniciar o jogo
Para iniciar, acesse o site [Jogo da Forca](https://guilhermesmotta.github.io/jogo-da-forca-js) ou realize o download do arquivo `.zip` contendo todo o código, extraia em uma pasta própria e abra o arquivo `.html` no seu navegador de preferência.
## Como jogar
Para jogar, insira uma letra na caixa de texto e clique no botão `Enter` no teclado ou no botão `Confirmar` na tela do jogo.
Para começar novamente, clique no botão `Reiniciar` na tela do jogo.
